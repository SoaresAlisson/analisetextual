7.1 Base plots

CAPÍTULO EM CONSTRUÇÃO

O R vem com um pacote básico, nativo de gráficos. Repetindo o exemplo da introdução ao R na seção sobre fatores, fazendo um gráfico de barras simples a partir de um vetor, mas agora explicando a parte gráfica e aproveitando o pipe do tidyverse.

library(magrittr)
## 
## Attaching package: 'magrittr'
## The following object is masked from 'package:purrr':
## 
##     set_names
## The following object is masked from 'package:tidyr':
## 
##     extract

aeroportos <- c("BSB", "CON", "BSB", "VIC", "GUA", "FOR", "MAO","GUA", "CON", "CON", "REC", "UDI", "VIC", "GUA")
# Numa versão mínima:
aeroportos %>% factor %>% summary %>% barplot

E numa versão mais elaborada

aeroportos %>%
  # transformando o vetor em fator para o barplot funcionar
  factor %>%
  # somando os repetidos
  summary %>%
  # ordendando com "sort()", e de modo decrescente com "decreasing = TRUE"
  sort(., decreasing = TRUE) %>%
  # plotando o gráfico
  barplot(
    main = "Aeroportos Brasil", sub = "Sublegenda", 
    xlab = "Legenda eixo x", ylab = "Legenda eixo y",
    legend.text = "Frequência",
    # cores 
    col = c("darkred", "indianred")
  )

7.4 O pacote ggplot2

O ggplot2 é um pacote da suíte de pacotes do tidyverse e segue a gramática dos gráficos de Hadley Wickham. Caso não tenha ainda instalado o ggplot2 ou o tidyverse. Instalando o pacote ggplot:

install.packages('ggplot2')

Carregando o pacote:

library(ggplot2)

Na lógica da gramática dos gráficos, O ggplot possui 7 níveis:

    1. dados
    1. aesthetics aes() , definindo os eixos x e y
    1. geoms, qual tipo de gráfico
    1. facets
    1. Estatística
    1. Coord ou Sistema de coordenadas
    1. Temas, theme: muda o fundo do gráfico.

Desse modo, vamos plotando o gráfico, nível por nível. A ordem pode mudar um pouco, alguns podem estar ausentes, mas são obrigatórios os três primeiros, e sem eles, não temos o gráfico. Vamos fazer um histograma, que é um gráfico simples, e só precisamos do eixo x.

ggplot(data = MeuDataBase, aes(x=ColunaDoDatabase_que_será_o_eixo_x)) + geom_histogram()
ggplot(data = mtcars, aes(x=mpg)) + geom_histogram()
